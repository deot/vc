## 提示框（Alert）

提示框.


### 属性

| 属性         | 说明                    | 类型        | 可选值                                  | 默认值     |
| ---------- | --------------------- | --------- | ------------------------------------ | ------- |
| modelValue | 是否显示, v-model         | `boolean` | -                                    | `true`  |
| type       | 警告提示样式                | `string`  | (`info` `success` `error` `warning`) | `info`  |
| title      | 标题                    | `string`  | -                                    | -       |
| desc       | 辅助性文字, 也可通过默认 slot 传入 | `string`  | -                                    | -       |
| icon       | 图表icon                | `string | boolean`  | -                                    | -       |
| closable   | 是否可关闭                 | `boolean` | -                                    | `false` |
