## 分割线（Divider）
区隔内容的分割线。

## API

### 属性

| 属性        | 说明         | 类型        | 可选值                       | 默认值      |
| --------- | ---------- | --------- | ------------------------- | -------- |
| vertical  | 分割线方向      | `boolean` | -                         | `false`  |
| placement | 设置分割线文案的位置 | `string`  | (`left` `right` `center`) | `center` |
